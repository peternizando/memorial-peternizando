# Release 1.4.0
Data: 07/07/2026

## Adicionado

- Navegação entre fotos no Lightbox.
- Botões de foto anterior e próxima.
- Contador de imagens.
- Navegação circular (primeira ↔ última imagem).
- Estrutura preparada para futuras evoluções do Lightbox.
- Melhorias de usabilidade Mobile First.
- Bloqueio da rolagem da página enquanto o Lightbox estiver aberto.

## Melhorado

- Organização do código do `lightbox.js`.
- Estrutura HTML do Lightbox.
- CSS da navegação.
- Comentários padronizados.

## Corrigido

- Ajustes no fechamento do Lightbox.
- Melhor tratamento do foco.
- Melhor controle do contador.
