# Memorial Peternizando Lite (V1)

## Objetivo

O Memorial Peternizando Lite (V1) é a primeira versão oficial da plataforma de memoriais digitais da Peternizando.


Seu objetivo é proporcionar uma experiência simples, elegante e emocionante através de um memorial digital acessado por NFC ou QR Code.

---

# Funcionalidades

## Acesso

- NFC
- QR Code
- Link direto

---

## Conteúdo

Cada memorial poderá possuir:

- Nome do pet
- Mensagem de homenagem
- Foto principal
- Até 10 fotos
- 1 vídeo

---

## Fotos

Quantidade:

- mínimo: 1
- máximo: 10

Formatos aceitos:

- JPG
- JPEG
- PNG

Recomendação futura:

- WEBP

---

## Vídeo

Quantidade:

- máximo: 1

Formato:

- MP4

Codec recomendado:

- H.264

Tamanho máximo:

- 30 MB

---

## Compartilhamento

Permitir compartilhar o memorial utilizando a funcionalidade nativa do celular.

Exemplos:

- WhatsApp
- Facebook
- Instagram
- Copiar link

---

## Hospedagem

GitHub Pages

---

## Estrutura

Cada memorial será totalmente independente.

Cada cliente possuirá sua própria pasta contendo:

- dados.json
- imagens
- videos

---

## Identificação

Cada memorial possuirá um identificador único.

Formato:

PTZ-XXXXXXXX

Exemplo:

PTZ-A8F4K9XM

---

## Tecnologias

- HTML5
- CSS3
- JavaScript
- JSON
- GitHub Pages

---

## Não faz parte da V1

- Login
- Cadastro de usuários
- Banco de dados
- API
- Área administrativa
- Comentários
- Música de fundo
- Linha do tempo
- Múltiplos vídeos
- Álbuns de fotos
- Upload de arquivos
- Inteligência Artificial

---

# Objetivo da V1

Validar a experiência do cliente.

O sucesso da V1 será medido pela facilidade de uso, desempenho, emoção transmitida e simplicidade na criação de novos memoriais.

---

# Filosofia

Uma homenagem digital deve emocionar.

Nunca deve parecer um sistema.
