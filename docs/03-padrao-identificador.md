# Padrão de Identificação dos Memoriais

## Objetivo

Padronizar a identificação única dos memoriais digitais da Peternizando.

---

# Prefixo Oficial

Todos os memoriais deverão utilizar o prefixo:

PTZ

---

# Máscara

PTZ-XXXXXXXX

Onde:

X = Letra maiúscula (A-Z) ou número (0-9)

Quantidade:

8 caracteres.

---

# Exemplos Válidos

PTZ-A8F4K9XM

PTZ-H7X2MPQ4

PTZ-Q9R4LW8N

PTZ-L8R3TYQ7

---

# Exemplos Inválidos

PTZ-123

PTZ-AAAAAAAAAAAA

PTZ-000001

PTZ-abcdef12

---

# Regras

- Cada memorial possui um único identificador.
- O identificador nunca poderá ser alterado.
- O identificador nunca poderá ser reutilizado.
- O identificador será utilizado:
  - Nome da pasta
  - URL
  - NFC
  - QR Code
  - Controle interno

---

# Estrutura

memoriais/

└── PTZ-A8F4K9XM/

---

# Observações

O nome do pet nunca deverá ser utilizado como identificador.
