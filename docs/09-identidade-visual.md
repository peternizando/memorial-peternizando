# Identidade Visual

## Objetivo

Este documento define a identidade visual oficial do Memorial Peternizando Lite (V1).

Toda interface do sistema deverá seguir estas diretrizes.

---

# Filosofia

O Memorial Peternizando não é apenas um site.

É uma homenagem digital.

Cada elemento visual deverá transmitir:

- Amor
- Respeito
- Saudade
- Gratidão
- Elegância

O usuário nunca deverá sentir que está utilizando um sistema.

Ele deverá sentir que está revivendo uma lembrança.

---

# Experiência

Ao abrir um memorial, a experiência deverá ser:

1. Simples.

2. Leve.

3. Elegante.

4. Emocionante.

5. Intuitiva.

---

# Paleta de Cores

## Cor principal

Branco

Representa:

- Paz
- Leveza
- Pureza

---

## Cor secundária

Cinza muito claro

Utilizado apenas como fundo.

---

## Cor de destaque

Preto

Utilizado em:

- Botões
- Títulos
- Informações importantes

---

## Cor de apoio

Cinza escuro

Utilizado em textos secundários.

---

# Tipografia

Fonte principal:

Sans-serif.

Prioridade:

- Segoe UI
- Arial
- Helvetica

A leitura deverá ser confortável em celulares.

---

# Espaçamento

Toda página deverá possuir bastante espaço em branco.

O conteúdo nunca deverá parecer apertado.

---

# Foto Principal

A foto principal é o elemento mais importante da página.

Regras:

- Grande
- Centralizada
- Formato circular
- Alta qualidade

---

# Nome do Pet

O nome deverá possuir maior destaque que qualquer outro texto da página.

Sempre centralizado.

---

# Mensagem

A mensagem deverá ser curta.

Fácil de ler.

Sempre centralizada.

---

# Seções

A ordem oficial da página será:

1. Foto Principal

2. Nome

3. Mensagem

4. ❤️ Recordações

5. 🎥 Uma lembrança especial

6. ❤ Compartilhar esta homenagem

7. Rodapé

---

# Recordações

Representa os momentos vividos com o pet.

Não utilizar o termo "Galeria".

Sempre utilizar:

❤️ Recordações

---

# Vídeo

Representa a principal lembrança em movimento.

Não utilizar:

Vídeo

Sempre utilizar:

🎥 Uma lembrança especial

---

# Compartilhamento

Texto oficial do botão:

❤ Compartilhar esta homenagem

---

# Rodapé

Sempre utilizar:

Peternizando

Transformando saudade em presença.

---

# Responsividade

O Memorial deverá ser desenvolvido inicialmente para dispositivos móveis.

Posteriormente deverá adaptar-se para tablets e computadores.

Mobile First é a estratégia oficial do projeto.

---

# Performance

A experiência deverá ser rápida.

Evitar bibliotecas desnecessárias.

Evitar animações pesadas.

Priorizar carregamento rápido.

---

# Acessibilidade

Todas as imagens deverão possuir atributo ALT.

Os textos deverão possuir bom contraste.

Os botões deverão possuir tamanho adequado para toque.

---

# Emoção

Toda decisão de interface deverá responder à seguinte pergunta:

"Esta tela parece um sistema ou uma homenagem?"

Se parecer um sistema, a solução deverá ser revista.

Se parecer uma homenagem, a solução está correta.

---

# Identidade Oficial

Após aprovação deste documento, esta identidade visual passa a ser considerada oficial para todas as versões do Memorial Peternizando Lite (V1).
